PENGGUNAAN TAG-TAG PADA HTML

Untuk memperoleh suatu tampilan yang menarik dalam dokumen HTML, diperlu-
kan kemampuan khusus salah satunya yaitu kemampuan untuk memformat/meng-
atur teks pada dokumen HTML



Untuk melakukan pengaturan teks tidak terlepas dari penggunaan tag-tag HTML seperti:
1.Tag < i >tulisan < /i > : Untuk menampilkan tulisan miring
2.Tag < strike > tulisan < /strike > : Untuk menampilkan tulisan bergaris tengah
3.Tag < b > tulisan < /b > : Untuk menampilkan tulisan menjadi tebal
4.Tag < small > tulisan < /small > : Untuk menampilkan tulisan menjadi lebih kecil dari ukuran normal
5.Tag < big > tulisan < /big > : Untuk menampilkan tulisan menjadi lebih besar dari ukuran normal
6.Tag < tt > tulisan < /tt > : Untuk menampilkan tulisan seperti tulisan mesin ketik
7.Tag < u > tulisan < /u > : Untuk menampilkan tulisan bergaris bawah
8.Tag < sub > tulisan < /sub > : Untuk menampilkan tulisan menjadi subscript
9.Tag < sup > tulisan < /sup > : Untuk menampilkan tulisan menjadi superscript
